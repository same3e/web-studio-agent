# Quickstart

Start with:

```text
Use product-studio.

I want to create a web application for freelance trainers.
Trainers should manage clients, create workout plans, and track progress.
I am not technical, so recommend the most suitable architecture.
```

`product-studio` first summarizes what it understands, such as trainer and client roles, the likely first journey, and a probable authenticated product. It then asks grouped questions only about missing scope, data, permissions, and success criteria.

It presents up to three technical options in product language, recommends one, and requests useful industry, visual, product, UX, or interaction references. Add them with `Use add-reference to analyze these URLs or screenshots for this project.`

Review up to three concepts. Approve explicitly, for example: `Approve concept 2; use the calmer motion from concept 1.` The same `product-studio` skill records approval and starts the approved implementation automatically unless you ask it to stop.
