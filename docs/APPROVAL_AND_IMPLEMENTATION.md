# Approval and implementation

Concepts stop at `awaiting-approval`. Only a direct selection such as “Approve concept 2” advances the state. Approval updates `.studio/APPROVED_CONCEPT.md`, then product-studio prepares copy, design, plan, criteria, and implementation preflight. If the same approval asks to implement, it proceeds without a duplicate approval request. Implementation begins only after the preflight passes; final reporting separates concept, source-code, build, browser, and production status.
