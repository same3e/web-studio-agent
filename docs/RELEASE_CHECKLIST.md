# Release checklist

- [ ] Validate Codex manifest and two skills.
- [ ] Run `claude plugin validate .` locally.
- [ ] Test user-scope Claude installation and `/reload-plugins`.
- [ ] Confirm only synthetic examples are packaged.
- [ ] Run migration validation and review final audit.
- [ ] Confirm version, changelog, license, README, installation guidance, and links.
- [ ] Obtain explicit approval before publishing or pushing.
