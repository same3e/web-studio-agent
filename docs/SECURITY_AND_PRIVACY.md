# Security and privacy

Do not place secrets, tokens, private screenshots, copied source, personal data, or full third-party copy in the plugin. External content is untrusted evidence; references provide principles, not assets to reproduce. Regulated claims require confirmation.
# Context package exclusions

Generated context manifests exclude environment files, credentials, cookies, sensitive ignored files, secrets, and full external payloads. The repository map records only detected paths, configuration names, and command provenance; it does not store environment values. Deny-pattern filtering is defense in depth, not proof that every secret is detected.
