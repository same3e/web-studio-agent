# EXPO_CROSS_PLATFORM_MOBILE

## Decision profile

Use for iOS/Android delivery where shared mobile code fits team and feature constraints. Baseline: Expo/React Native, backend API, mobile auth, release tooling; validate device navigation, permissions, network recovery, accessibility. Do not use when critical native modules/performance require native-first architecture. Limitation: store release and native escape-hatch work.
