# EXISTING_PROJECT_INTEGRATION

## Decision profile

Use when an existing repository is the system of record. Baseline: preserve current stack, inspect manifests/code/tests, add only necessary modules; validate existing and new checks plus regression path. Do not use to replace a valid stack for preference alone. Limitation: inherited architecture and deployment constraints.
