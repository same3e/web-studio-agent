# Phase 4 black-box workflow evidence

The required ordinary-user prompts are recorded here for a host that can invoke the installed skills in a fresh chat. This execution environment cannot start the Codex CLI and has no Claude CLI, so no transcript is represented as runtime evidence. Deterministic routing, preflight, reference-library, and release-gate coverage is executed by `scripts/test_phase4.ps1`; it is not a substitute for black-box agent execution.
